# @gkd-kit/inspect

- <https://i.gkd.li> github pages

GKD 网页端审查工具,

- 抓取快照
- 审查快照
- 分享快照
- 连接调试 GKD APP 内部 HTTP Server
- 实时启用订阅规则, 实时执行点击选择器
- 选择器路径视图
